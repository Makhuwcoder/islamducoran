## Partie I · Les racines

{% include nl.html racine="ح · ل · ل" contenu="
**Al-Farāhīdī** : *al-ḥill* — la levée de l'interdiction, ce qui est ouvert, accessible, non lié. *Ḥalla* — il s'est dénoué, il est devenu licite.

**Ibn Fāris** : sens primitif — *al-fatḥ wa-l-inḥilāl* — l'ouverture et le dénouement. *Ḥalāl* : ce qui est délié, ouvert, accessible.
" %}

{% include nl.html racine="ح · ر · م" contenu="
**Al-Farāhīdī** : *al-ḥurma* — ce qui est protégé, inviolable, dont l'accès est interdit. *Al-ḥirmān* — la privation, l'exclusion.

**Ibn Fāris** : sens primitif unique — *al-manʿ* — l'interdiction, la mise hors d'atteinte.
" %}

## Partie II · Qui a le droit de déclarer ḥalāl ou ḥarām ?

### § 1 · Le principe coranique fondamental

{% include verset.html ref="Sourate 16 · v. 116" ar="وَلَا تَقُولُوا لِمَا تَصِفُ أَلْسِنَتُكُمُ الْكَذِبَ هَٰذَا حَلَالٌ وَهَٰذَا حَرَامٌ لِّتَفْتَرُوا عَلَى اللَّهِ الْكَذِبَ" trl="*Wa-lā taqūlū… hādhā ḥalālun wa-hādhā ḥarāmun li-taftarū ʿalā llāhi l-kadhib*" trad="Ne dites pas, selon ce que vos langues décrivent mensongèrement, « ceci est *ḥalāl* et ceci est *ḥarām* », pour **forger sur Allaah le mensonge**." %}

{% include callout.html type="dit" label="Dit — monopole législatif d'Allaah" content="Déclarer *ḥalāl* ou *ḥarām* sans autorisation coranique est qualifié d'*iftirāʾ ʿalā llāh* — forger un mensonge sur Allaah. Ce n'est pas une exagération rhétorique : c'est la formule coraniqe du *shirk* (cf. S.4:48). Seul Allaah déclare *ḥarām*." %}

{% include verset.html ref="Sourate 6 · v. 119" ar="وَقَدْ فَصَّلَ لَكُم مَّا حَرَّمَ عَلَيْكُمْ" trl="*Wa-qad faṣṣala lakum mā ḥarrama ʿalaykum*" trad="Il a **détaillé pour vous ce qu'Il vous a interdit**." %}

{% include callout.html type="pivot" label="Verset pivot — ce qu'il implique" content="*Faṣṣala* — Il a détaillé, Il a explicité. Le Coran affirme que les interdictions ont été détaillées. Ce qui n'est pas dans ce détail n'est pas interdit. L'inventaire des interdictions coraniques explicites est donc **fermé** — non ouvert à l'addition par d'autres sources." %}

## Partie IV · Le principe de la permission par défaut

{% include verset.html ref="Sourate 2 · v. 29" ar="هُوَ الَّذِي خَلَقَ لَكُم مَّا فِي الْأَرْضِ جَمِيعًا" trl="*Huwa lladhī khalaqa lakum mā fī l-arḍi jamīʿan*" trad="C'est Lui qui a créé pour vous **tout ce qui est dans la terre**." %}

{% include callout.html type="inf" label="Inférence légitime" content="La création a été faite *pour vous* (*lakum*) — tout ce qui est dans la terre. Couplé à S.6:119 (*Il a détaillé ce qu'Il a interdit*), l'inférence est : ce qui n'est pas explicitement interdit dans le texte est disponible. C'est la règle de la permission par défaut (*al-ibāḥa al-aṣliyya*) — inférence forte depuis la cohérence du corpus." %}

## Partie V · Synthèse

{% include callout.html type="dit" label="Trois principes textuels" content="
1. Seul Allaah déclare *ḥarām* — tout autre déclarant forge un mensonge sur Allaah (16:116)
2. Les interdictions ont été détaillées (*faṣṣala*) dans le Coran (6:119)
3. La création est mise à disposition des hommes — ce qui n'est pas interdit est disponible (2:29)
" %}

{% include callout.html type="nondit" label="Non-dit" content="Le Coran ne donne aucun mandat à qui que ce soit pour étendre la liste des interdictions au-delà de ce qu'il a lui-même énoncé. Les milliers d'interdictions supplémentaires produites par le fiqh au cours des siècles n'ont pas de fondement textuel coranique." %}
